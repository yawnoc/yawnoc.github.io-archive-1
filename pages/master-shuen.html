Introduction

[▼ Translation]

《孫子算經》, or "Master 孫 (Shün) (Sun) (Sun)'s Computational Classic", like the Sea Island Computational Classic, is one of the ten canon classics of Chinese mathematics. Probably written in the 3rd to 5th centuries AD, the author 孫子 (Shün Chee) (Sun Tzŭ) (Sun Zi) is not the same 孫子 (Shün Chee) (Sun Tzŭ) (Sun Zi) who wrote The Art of War.

In late April 2019, I set myself the rather ambitious task of translating the entire 《孫子算經》 into English. Apparently this has been done before, but I shall not read that translation (which is behind a paywall anyway) until I have completed mine.

Each night I will be translating a few sentences, maybe 50 characters. Since the text is some 9000 characters long, the whole thing will take around 9000/50 = 180 days, so perhaps I can complete it by the end of this year.

Versions

There are four versions which I refer to, with slight differences in wording between them. Unless otherwise noted I will be following the text from Version D. The punctuation is based on that in Version C (ctext.org), but with changes to my liking. I have grouped the paragraphs mostly following the line breaks of Versions A, B and D, with groupings which in my opinion are more logical than those in Version C.

A quick system for referring to specific pages is as follows:

A note on mass and weight

Classical Chinese does not distinguish between mass and weight; nor is this a physics text in which the distinction is important. Therefore I will be using the two terms interchangeably.

A note on thousands and myriads

In Chinese, numbers are grouped in 萬, myriads (10000s), rather than thousands (1000s). For example, 八萬五千 (85000) is literally "eight myriad five thousand" rather than "eighty-five thousand". However, I shall default to rendering numbers in the English using the thousands grouping, only using the myriads grouping for passages where the thousands grouping is awkward.
